## YuYu-清酒零售商城-NestJs-Api
一个开源YuYu-清酒零售商城Api服务端，项目集成Nestjs+Mongodb+Jwt等。
项目是前后端分离，分为三端：Api服务端，Admin管理端，Web前端(uniapp),


## 

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
